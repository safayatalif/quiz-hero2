ForEach: ForEach is used to loop through an array and execute a function on each element in the array. It does not return a value.
Map: Map is used to loop through an array, modify each element in the array, and return a new array.
Filter: Filter is used to loop through an array and filter out elements based on a condition. It returns a new array with the elements that match the condition.
Find: Find is used to loop through an array and find the first element that matches the condition. It returns the value of the element that matches the condition.

Local Storage: Local Storage is a type of web storage that allows websites and web apps to store and access data in the browser. Local Storage data is stored in the browser until it is manually cleared. It is more secure than cookies .
Session Storage: Session Storage is similar to Local Storage, except that data stored in Session Storage is only available for the current page and browser session. The data stored in Session Storage is cleared when the browser tab or window is closed.
Cookies: Cookies are small text files that are stored on the user's computer by the web browser. They contain small pieces of data, such as site preferences, and are used to store information such as login details.

The event loop is a process that allows JavaScript code to run asynchronously, meaning that it can run in the background while other code is executing. In this way, it helps to ensure that the program can respond to user events in a timely manner.

The quizTimer() function is used to set a timer for a quiz, display the remaining time in minutes and seconds, and change the color of the timer to red when the time left is less than one minute. It also takes a dismiss parameter which can be used to stop the timer if needed.
