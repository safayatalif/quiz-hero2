DOM (Document Object Model) is a programming interface for web documents. JavaScript does not understand HTML elements, which is where the DOM comes to the rescue and uses it to manipulate JavaScript HTML elements.Browsers use the DOM to display web pages, and the DOM can be accessed and manipulated by JavaScript.

There are 5 ways select elements in a DOM use selector.
  • getElementById()
  • getElementsByClassName()
  • querySelectorAll()
  • getElementsByTagName()
  • querySelectorAll()

Event bubbling is a mechanism in JavaScript, where an event is triggered on a specific element and then propagates to its parent element and elements beyond. For example: When a button is clicked, the click event will be triggered first on the button, then on its parent element and on to its outer elements.

A callback function is a function that is passed as an argument to another function. It is used to run it later. Example : "function print(callback) { callback(); }"
